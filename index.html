Tableau Mastery:
Data Visualization Project for Airbnb Insights

In this Tableau project, I aimed to derive insights to enhance Airbnb's business using a dataset downloaded from Kaggle. Due to Tableau Public's limitations, I filtered the dataset to 10 million rows from the original 23 million. I joined multiple datasets based on common IDs and created various visualizations. This project improved my skills in creating effective data visualizations, utilizing functions, filters, charts, and joins in Tableau.
